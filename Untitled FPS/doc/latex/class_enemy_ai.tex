\doxysubsection{Detailed Description}
Base class for controlling enemies. 

Every specific \char`\"{}\+Ai\char`\"{} class inherits this one. Determines the actions taken by an enemy based of the distance to the player. Also controls basic functionality like taking damage, dying etc.
