Detailed Description

Responsible for spawning elements at different transforms existing in dungeon parts.

Those elements can be: adjacent rooms/corridors, doors, walls.

int  parentEntryPoint,
int  partType 
)

Used to create new parts of the dungeon.

Spawnes a prefab of a new room/corridor at the location stored in the entryPoint variable. Then uses it to create an EntryPoint type obejct.

Parameters
parent- the DungeonPart object that the newly spawned part is connected to
parentEntryPoint- which of the parent's EntryPoint variables is the newly spawned part connected to
partType- what type of DungeonPart is the parent object- room or corridor. If the parent part is a room the spawned part is going to be a corridor and vice versa
Returns
The newly spawned DungeonPart object
